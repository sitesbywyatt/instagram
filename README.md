# instagram

A collection of UI components and experiments.

## GitHub Pages

This repo can be published as a GitHub Site. To enable:

1. Go to **Settings** → **Pages**
2. Under **Source**, choose **Deploy from a branch**
3. Select branch **main** and folder **/ (root)**
4. Save — your site will be at `https://<your-username>.github.io/instagram/`

## Sections

- **[Loaders](loaders/)** — Spinners, skeleton, progress bars, dot bounce, circular progress
- **[CSS Spinners](css_spinners/)** — 10 pure CSS loading indicators
- **[Buttons](buttons/)** — 24 buttons: hover, click, ripple, magnetic, submit states, neumorphic
- **[Text Effects](text_effects/)** — Typewriter, gradient, glitch, reveal, scramble, wavy
- **[Cards](cards/)** — 3D flip, hover lift, glassmorphism, tilt, expand
- **[UI Components](ui_components/)** — Toggles, checkboxes, hamburger, dropdowns, tooltips, toasts
- **[Background Effects](background_effects/)** — Gradients, particles, floating shapes, noise, aurora
- **[Data Visualization](data_visualization/)** — Counters, skill bars, pie charts, line graphs
- **[Micro-interactions](micro_interactions/)** — Like, add to cart, input focus, error shake, copy, star rating
- **[Image Effects](image_effects/)** — Hover zoom, before/after slider, reveal, tilt, lazy blur
- **[Scroll Animations](scroll_animations/)** — Fade in, slide in, sticky header, progress bar, snap
